# Modern TypeScript Application Boilerplate

# Quick Setup

```
npm install -g firebase-tools
npm install
```

* Run DevServer for Development
```
npm start
```

* Run Emulation server
```
npm run serve
```

* Deploy
```
npm run deploy
```
